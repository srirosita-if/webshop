# Webshop - Aplikasi Pemesanan Produk Online

Proyek ini merupakan tugas mata kuliah Pemrograman Web yang bertujuan untuk menerapkan konsep dasar pembuatan aplikasi berbasis website. Aplikasi yang dibuat merupakan toko online sederhana bertema produk pink dengan proses pemesanan yang terdiri dari pengisian data pelanggan dan konfirmasi pesanan.

## Deskripsi Sistem
Aplikasi ini menyediakan fitur pemesanan produk secara online dengan menampilkan daftar produk dan pengguna dapat melakukan pemesanan melalui form pemesanan. Data pemesanan akan ditampilkan kembali dalam bentuk halaman konfirmasi sehingga pembeli dapat memeriksa kembali pesanan sebelum melanjutkan proses.


## Fitur Sistem
- Menampilkan data produk
- Form pemesanan produk
- Proses konfirmasi data pesanan
- Tampilan antarmuka bertema warna pink

## Cara Menjalankan Program
1. Siapkan folder project di dalam direktori `htdocs`.
2. Aktifkan Apache pada aplikasi server lokal.
3. Jalankan aplikasi melalui browser dengan alamat: (http://localhost/webshop/index.php)
